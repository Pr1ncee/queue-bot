<h1>Queue-bot</h1>

A application for queue creating in Telegram.
<ul>
<li>Written in Python;</li>
<li>Using pyTelegramBotAPI.</li>
</ul>
There are two possible subgroups in QueueBot.<br>
An user can
<ol>
<li>Create a queue;</li>
<li>Close a queue;</li>
<li>Join a queue;</li>
<li>Get out of a queue.</li>
</ol>
